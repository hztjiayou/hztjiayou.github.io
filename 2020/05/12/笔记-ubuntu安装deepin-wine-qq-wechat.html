[笔记]ubuntu安装deepin-wine-qq/wechat

写在前面

这是一篇老生常谈的笔记，用以归纳总结知识，方便日后查找。本文记录了ubuntu20.04如何安装wine-qq和wine-wechat和安装过程中遇到种种问题及解决方法。

安装步骤

  • 安装 deepin-wine-ubuntu
  • 下载相关应用容器（qq/wechat等）

安装deepin-wine-ubuntu

项目地址：https://github.com/wszqkzqk/deepin-wine-ubuntu

安装方法

  • 克隆github项目到本地

  • 解压后切换到解压目录，授予 install.sh可执行权限，执行： ./install.sh

  • 完成安装

安装相关容器

容器下载地址：http://mirrors.aliyun.com/deepin/pool/non-free/d/

安装QQ

可以选择如下地址下载，选择.ded文件下载安装，推荐使用dpkg -i命令安装：

TIM
QQ
QQ轻聊版

问题：

  • 如果安装系统的时候选择的是英文，那么会有后期显示中文乱码，具体解决方法未知。

  • 如何把QQ、wechat的系统托盘集成到状态栏?

    • 安装gnome-tweak-tool

      sudo apt install gnome-tweak-tool

    • 安装top-icons-plus扩展插件

      sudo apt-get install gnome-shell-extension-top-icons-plus gnome-tweaks

    • 安装完成后重启系统，在系统的所有应用程序中会有一个“扩展（Extensions）”，点击“扩展”，打开 Topicons Plus 即可

安装wechat

容器下载连接：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.wechat/

在此连接中有两个deb安装包，我在尝试 deepin.com.wechat_2.6.8.65deepin0_i386.deb 这个版本后，出现安装错误，于是安装了另一个版本，安装正常。

问题

  • wechat无法发送图片

    sudo apt install libjpeg62：i386
